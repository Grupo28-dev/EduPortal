Curso de HTML y CSS

Aprendé a maquetar sitios modernos, accesibles y mobile‑first desde cero.

  • Comprender HTML semántico y buenas prácticas de estructura.

  • Optimizar accesibilidad, performance y SEO técnico básico.
Programa
